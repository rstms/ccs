Welcome to ccs's documentation!
===============================

**CCS** is a command line interface for the CloudSigma API.

It implements a set of commands with arguments and options representing the functionality of the excellent pycloudsigma module at the command line.  The purpose is to facilitate creation and management of cloudsigma resources at the command line and in shell scripts.
